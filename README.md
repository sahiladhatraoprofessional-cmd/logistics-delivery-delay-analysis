## Project Overview
This project analyzes an e-commerce logistics dataset to identify the key factors driving delivery delays

The analysis was conducted using SQL for data exploration, Python for deeper analytical insights, and Power BI for interactive dashboard visualization.

## Problem Statement
The company observed increasing delivery delays. The objective of this project was to identify the root causes behind these issues and provide insights for operational improvement.

## Tools Used
- SQL (MySQL)
- Python (Pandas)
- Power BI

## Dataset
The dataset contains 5000 orders with the following fields:

- Order ID
- Order Date
- Ship Date
- Delivery Date
- City
- Region
- Warehouse
- Product Category
- Shipping Mode
- Distance
- Shipping Cost
- Order Value

## Key Analysis Steps
1. SQL analysis to identify operational patterns.
2. Python analysis to create delivery metrics and analyze relationships.
3. Power BI dashboard to visualize logistics performance.

## Key Insights
- West region showed the highest delivery delay.
- Warehouse W3 had the slowest delivery performance.
- Express shipping reduced delivery time but increased cost.
- Shipping cost strongly increased with distance.

## Business Recommendation
Improving operational efficiency at Warehouse W3 and optimizing logistics in the West region could significantly reduce delivery delays.

## Dashboard
The Power BI dashboard visualizes delivery performance across regions, warehouses, and shipping modes.
